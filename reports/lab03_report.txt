P2: Pipeline Thinking
Explain:
• why ingestion must precede preprocessing:
Ingestion acts as a "trust filter" that verifies data structure and accessibility before performing any numerical transformations.

• what happens if invalid data is not removed:
Placeholder values like INVALID cause mathematical errors, script crashes, and biased results in downstream analysis.

• how ingestion affects model quality:
Proper ingestion ensures models are trained on clean, verified data, which is essential for reliable predictions and system integrity
